The First
Baltimore Hackathon

  • Build.

  • Meet People.

  • Have Fun.

What is this?

Build a hardware or software project from idea to prototype in a weekend!

You can participate individually or as a team.

When?

November 19-21 2010. The event will start at 5pm on Friday 11/19 and end at 7pm on Sunday 11/21

Where?

It will be held at Beehive Baltimore, 2400 Boston St.


Rules

The rules will evolve via this Google Doc over the next month or so. We encourage you to add to that doc and ask questions via the doc. The rules will be considered final when opening ceremonies begin.

Awards

Judges will be recognizing the best projects in these categories:

  • Best Software Prototype - Individual
  • Best Software Prototype - Group
  • Best Hardware Prototype - Individual
  • Best Hardware Prototype - Group
  • Audience Favorite

How do I participate?

You can register for the event on its EventBrite page
